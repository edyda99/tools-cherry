How to use the password generator

Pick Random password for the strongest, most compact option, or Passphrase when you need something you can type and remember. For a random password, drag the length slider — 16 characters or more is a good target — and tick the character types you want to include. A new password is generated instantly every time you change a setting, and you can press Generate new password for a fresh one. Use the Copy button to put it on your clipboard.

The coloured bar shows an estimate of the password's strength, measured in bits of entropy — roughly, how many guesses an attacker would need. More length and more character types both push the entropy higher. The "avoid ambiguous characters" option drops look-alikes such as the letter O and the number 0, which is handy when a password has to be read aloud or typed from a screen.

What makes a password strong?

Strength comes from length and randomness, not from clever substitutions like swapping an "a" for an "@". A truly random 16-character password drawn from a large character set is far harder to crack than a short one peppered with symbols. Avoid dictionary words, names, dates, and anything tied to you personally, and never reuse the same password across accounts — if one site is breached, every account sharing that password is at risk.

If you struggle to remember passwords, that is exactly what a password manager is for: it stores a unique strong password for every account so you only have to remember one master passphrase. This generator pairs naturally with that workflow — generate a strong password here and save it in your manager.

Every password is created locally with your browser's cryptographically secure random number generator. Nothing you generate is uploaded, logged, or stored, and the page keeps working even if you go offline.
