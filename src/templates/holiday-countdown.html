How to use the holiday countdown

Tap any holiday button to see how long is left until it arrives. The big number is the days remaining; the boxes beneath it count the hours, minutes, and seconds in real time.

Want a different occasion — a birthday, a trip, a deadline? Use the date picker to count down to any date you choose.

When a holiday has already passed this year, the countdown automatically jumps to next year's date, so you always see the next time it comes around.

Everything runs in your browser using your device's clock — nothing is uploaded.

Common countdown questions

How many days until Christmas? Tap the Christmas button to see a live count to December 25. After Christmas, it rolls forward to next year on its own.

When is Thanksgiving? Thanksgiving falls on the fourth Thursday of November, so the date changes each year — the countdown works it out for you.

When is Memorial Day? Memorial Day is the last Monday of May. Like Thanksgiving, it moves each year, and the tool calculates the right date automatically.

Why does Easter move around? Easter is set by the lunar calendar, so it lands on a different Sunday each spring. This countdown computes the correct Easter date for any year.

How do I count the days between today and a future date? Pick the date in the picker — the headline number is exactly how many whole days are left.
